# Cyclos - Proven Local Currency Infrastructure

## Overview

Cyclos is state-of-the-art digital payment software used by **3,500+ LETS, Timebanks, and social currency systems** worldwide. It provides production-ready infrastructure for community currencies.

**Website:** https://www.cyclos.org/
**Free Version:** Cyclos 4 Communities
**License:** Proprietary (free tier available)

---

## Why Cyclos Matters

Unlike experimental systems, Cyclos has:
- **10+ million users** across various implementations
- **Proven track record** (Bristol Pound, Calgary Dollars, etc.)
- **Mobile apps** for Android and iOS
- **NFC and QR code** payment support
- **Multi-currency** management

---

## Key Features

### Member Functions
- Message each other
- Share talents and skills
- Offer and request goods/services
- Register exchanges
- Organize events
- View transaction history

### Payment Methods
| Method | Description |
|--------|-------------|
| Web Portal | Browser-based account access |
| Mobile App | Android/iOS applications |
| NFC Card | Contactless physical card payments |
| QR Code | Scan-to-pay functionality |
| SMS | Text-based transactions |

### Administrative Features
- Multiple currency network management
- Inter-network connections
- Configurable credit limits
- Transaction reporting
- Member verification

---

## Technical Specifications

### Architecture
- **Language:** Java
- **Database:** PostgreSQL
- **Security:** ISO 27002 compliant
- **Transactions:** ACID compliant
- **Authentication:** SHA-256

### Deployment Options
1. **Cyclos 4 Communities** - Free, self-hosted
2. **Cyclos 4 PRO** - Full features, requires license
3. **Cyclos Cloud** - Managed hosting

## Real-World Implementations

### Calgary Dollars (Canada)
- Uses Cyclos for digital infrastructure
- iOS/Android app with simple UI
- Features: balance, payments, marketplace ads
- Community services integration

### Bristol Pound (UK)
- One of UK's largest local currencies
- Cyclos-powered digital transactions
- Integration with local businesses

## Relevance to TechnoCommune

### Strengths
- **Production-ready** - No need to build from scratch
- **Compliance-aware** - Designed for regulated contexts
- **User-friendly** - Proven UX for non-technical users
- **Multi-modal** - Web, mobile, NFC, QR, SMS

### Weaknesses
- **Not fully open source** - Core is proprietary
- **Server-dependent** - Requires internet connectivity
- **Centralized** - Single database architecture
- **Not mesh-capable** - No offline-first design

### Potential Use Cases
1. **Rapid MVP** - Get trading quickly
2. **User interface reference** - Study proven UX
3. **Hybrid approach** - Cyclos frontend + custom backend
